# Basic prerequisites

To work with Vanadium in Java, we recommend that you have:

* Java 7 or above
* [Gradle]

# Getting the Vanadium libraries

The Vanadium Java libraries are distributed using [Maven].

[Gradle]: http://gradle.org/
[Maven]: http://maven.apache.org/
